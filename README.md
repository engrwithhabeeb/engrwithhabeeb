# 👋 Hi, I’m Habeeb — Creator of TipDevs  

## 🚀 My Tech Journey  
I started as a complete beginner, driven by curiosity and a passion to create.  
Through *The Odin Project*, I mastered HTML & CSS and dove deep into JavaScript —  
building projects like a **modular to-do list app** with Webpack, PubSub architecture, and localStorage.  

I enjoy working with APIs, building responsive interfaces, and keeping my code clean with ESLint & Prettier.  
I’m also building **TipDevs**, a “Tech Brainiac Community” for sharing knowledge, showcasing projects,  
and collaborating with other developers.  

💡 *Always learning, always building.*
